Promise são promesas que tem uma função booleana, ou seja sim ou não como resposta, onde sim ela recebe a caracteristica resolvida enquanto não recebe a caracteristica rejeitada, em termos mais simples, imagine um sistema de correios onde a compra do produto é uma promise, se ela chegar na sua casa então ela foi resolvida, se não ela foi rejeitada e por isso, as Promises podem ser consideradas um sistema Assincrono, ou seja, depende de uma variavel; Then() é um método que retorna uma promise resolvida, usando de argumentos tanto o callback quanto as funções; catch() já é um oposto de Then no quesito uso, se o then funciona com uma promise resolvida, o catch funciona caso seja rejeitada;Finally é o ultimo método de uma promise onde ela dá um veredito final sobre o resultado idependendente se deu resolvida ou rejeitada

No JavaScript, temos caracteres que representam algo e facilita no entendimento do código

Caracteres e seus significados

  • o sinal de igual (=) representa atribuição, usado para atribuir uma caracteristica a um objeto
  • Com o ponto final(.) acessamos o objeto podendo manipular suas caracteristicas dessa forma
  • os paragrafos ( ) são usados para convocação de um elemento, onde caso seja um texto, deve ser necessario o uso das aspas simples
  • Os Colchetes { } tem quase a mesma função do igual(=) pois seu objetivo também consiste em atribuir, mas invés de atribuir caracteristicas, colocamos outros comandos que fazem parte, nem todos os comandos utilizam deste caractere
  • == está relacionado ao verdadeiro sinal de igual, sendo um caractere de comparação invés de atribuição
  • === já com 3 sinais de iguais ele faz uma comparação mais especifica onde um valor 1 não seria igual a um texto 1
  • agora que sabemos as relações de igualdade, o caractere que denomina a diferenciação é a exclamação seguida de um sinal de igual (!=)
  • para representar o acréssimo "e" usamos duas vezes o sinal de E comercial (&&), por exemplo, if(condição 1 && condição dois) isso para fazer algo em ambas as condições
  • para somar ou subtrair algo, um valor por exemplo, usamos o sinal de mais ou menos, de forma bem simples
  • da mesma forma que o ponto final serve para acessar um objeto, podemos usar esses colchetes [ ] para acessar uma Array
  • existe uma forma simplificada de chamar uma estrutura de if, um elemento terceario representado por ?: onde o ? seria o equivalente ao if e o : seria o else assim simplificando a estrutura para uma unica linha

Lista de comandos

  • Alert = Abre uma janelinha de alerta e o que tiver dentro dos parenteses com aspas será o texto exibido
  • Function = Esse é um comando de alta prioridade que serve para criar funções e assim criar interações do usuario com a pagina, a estrutura do Function é simples, primeiro o nome do comando, no caso, function, espaço e o nome dado a função e por fim abre e fecha colchetes, lembrando bastante a estrutura de CSS onde tudo o que estiver dentro dos colchetes será somado ao nome da função
  • const = cria uma referencia constante cujo o valor fixo, ou seja, não é alterada (só para leitura)
  • let = cria uma referencia variavel
  • QuerySelector = esse comando é responsavel por puxar um elemento, selecionar o elemento (as referenciando dentro dos parenteses e aspas) da mesma forma que são referenciadas no CSS, classe com ponto, types de input com [ ]colchetes, id com hashtag e tags com o nome sem modificações no inicio, mas para isso ela é necessario indicar exatamente onde ela ira buscar, para isso é utilizado document.QuerySelector que ai sim ele irá buscar esse elemento no documento do projeto, logo após dizer o que será selecionado pelo querySelector, se colocarmos um ponto, poderemos ter acesso a todas as funcionalidades do elemento
  • QuerySelectorAll = tendo o mesmo objetivo que o querySelector, o querySelectorAll se difere por pegar todos os elementos que tenham o mesmo seletor, enquanto o querySelector, sempre pega o primeiro elemento com o seletor
  • addEventListener = Registra uma única espera de evento em um único alvo por exemplo click pra quando alguem clicar sobre o elemento com registro
  • While() = comando responsavel por repetições, muito util para facilitar o entendimento do código e para que não haja linhas desnecessarias com repetições de um mesmo código, sua funcionalidade é usada da seguinte maneira, dentro dos parenteses, usamos uma referencia na qual definira o fim do código, na aula, usamos o contador, onde contador era igual a 0, deixando da seguinte maneira (contador < 9) onde o contador era menor que 9, logo abaixo, dentro do while que por sua vez englobava usando os { } colchetes, colocamos o elemento que tenha a referencia, na aula foi usado o listaDeTeclas[contador].onclick = tocaSomPom e no final para que haja uma repetições dos outros sons, marcados numericamente com outros numeros de 0 a 9 foi colocado um novo valor da nossa referencia, onde o contador = contador + 1, assim o contador era somado com +1 isso até o numero 9 onde o while deixaria de receber o valor TrueUm alerta sobre esse tipo de comando, certifique-se de colocar um limitador nos parenteses, caso contrario, o que o laço de repetição englobar pode acabar não tendo o fim e ocasionando em um travamento da pagina, ou em piores caso, um travamento no próprio computador
  • console.log = esse comando é um util para receber informações diretas no console sem a necessidade de ficar digitando no DevTools, como por exemplo ver o valor de uma variavel
  • for = sendo também um comando de laço de repetição, sua estrutura se assemelha ao do While, com a diferença de que o for pode chamar referencia e o incredimento dentro dos próprios parenteses, deixando assim, muito mais organizado as linhas do código economizando no quesito quantidade
  • if = esse comando classico de toda linguagem, se apresenta no JS de forma bem simples, if(){} onde a condição dela fica entre os parenteses. A fim de economizar linhas no código, podemos declarar mais de uma condição usando o pipe(|) duas vezes, deixando a linha do código da seguinte maneira if(condição 1 || condição 2)
  • getElementBy = busca um elemento por uma caracteristica especifica
  • indexOf() = acessa o índice de um elemento, no caso o array dele
  • Push()/unshift() = o push adiciona um elemento ao final do array, já para adicionar no inicio do array, podemos usar o unshift
  • pop()/shift()/splice() = pop remove um elemento do final de um array e para retirar do inicio, podemos usar o Shift, mas caso necessario, tem como remover do índice usando o splice, para a remoção usamos o numerador do indice
  • Return this = Quando em modo estrito (strict mode), this mantém o valor undefined em funções globais e em funções anônimas que não estão vinculadas a nenhum objeto.
  • window = O window representa uma janela que contém o elemento DOM da página acessada. No objeto window é definida todas variáveis globais e funções que são executadas em um navegador.
  • preventDefault() = ele faz uma interrupção de algo, preveninto o padrão do código
  • createElement() = esse comando é responsavél por criar um novo elemento no html colocando a tag dentro dos parenteses
  • appendChild = Adiciona um nó ao final da lista de filhos de um nó pai especificado. Se o nó já existir no documento, ele é removido de seu nó pai atual antes de ser adicionado ao novo pai.
  • innerHTML = é como uma chave de acesso para o JS fazer mudanças no HTML
  • localStorage.setItem = adiciona um item no local storage, onde as informações ficam armazenadas
  • Fetch() = fornece uma interface via JS para acessar e manipular partes de uma API
  • Filter = Filtrar elementos em especifico de uma array
  • Map = é um método usado para chamar um callback em uma array, ele refaz uma array seguindo modificações, por exemplo aplicar um desconto em uma array de preços
  • Sort = esse comando é responsavel por organizar uma array, ele por padrão utiliza uma ordem unicode, mas com um ajuste do comando return e uma função podemos ajeitar a ordem da forma que quisermos
  • ForEach = tendo uma tradução de para cada, ele aplica uma função para cada elemento de uma Array
  • Reduce = esse método pega TODOS os valores de um array e transforma em apena um unico, podendo ser excluindo outros valores, isolando o unico valor, somando para gerar um valor novo e assim por diante, depende muito do que for feito
