Caderno JavaScript

Assim como o CSS, acrescentar o JavaScript é necessario usar um arquivo externo e ter como formato JS

Lista de comandos

  • const = cria uma variavel cujo o valor fixo, ou seja, só para leitura
  • Document.querySelector = Retorna o primeiro elemento dentro do documento (seja uma classe, ou um conteudo) que corresponde ao grupo especificado de seletores.
  • addEventListener = Registra uma única espera de evento em um único alvo por exemplo click pra quando alguem clicar sobre o elemento com registro
